拷贝控制习题

## 练习 13.1：拷贝构造函数是什么？什么时候使用它？

**【出题思路】**

理解拷贝构造函数的基本概念。

**【解答】**

如果构造函数的第一个参数是自身类类型的引用，且所有其他参数（如果有的话）都有默认值，则此构造函数是拷贝构造函数。

```cpp
class Foo {
public:
    Foo(const Foo& other);  // 拷贝构造函数
};
```

拷贝构造函数在以下几种情况下会被使用：

1. **拷贝初始化**（用 `=` 定义变量）
```cpp
Foo a;
Foo b = a;  // 调用拷贝构造函数
```

2. **将对象作为实参传递给非引用类型的形参**
```cpp
void func(Foo f);   // 传参时调用拷贝构造函数
func(a);
```

3. **返回类型为非引用类型的函数返回一个对象**
```cpp
Foo func() {
    Foo f;
    return f;   // 返回时调用拷贝构造函数
}
```

4. **用花括号列表初始化数组中的元素或聚合类中的成员**
```cpp
Foo arr[] = {a, b, c};  // 每个元素都调用拷贝构造函数
```

5. **初始化标准库容器或调用其 insert/push 操作时**
```cpp
vector<Foo> v;
v.push_back(a);   // 容器对元素进行拷贝初始化
```
